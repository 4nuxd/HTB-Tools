# 🛠️ HackTheBox Custom Tools

A curated collection of custom scripts and tools developed while solving **Hack The Box (HTB)** machines and challenges.

Some HTB rooms require writing small automation scripts, exploit helpers, or custom enumeration tools — this repository contains those utilities.

---

## ⚠️ Disclaimer

These tools are created for **educational purposes only**.

* Use them only in legal environments.
* Do not use against systems without proper authorization.
* I am not responsible for misuse.

---

## 📂 Repository Structure

```
.
├── enumeration/
├── exploitation/
├── privilege-escalation/
├── web/
├── crypto/
└── misc/
```

Each folder contains tools/scripts related to specific phases of penetration testing.

### 🔹 Enumeration Tools

## 🚀 Usage

Most scripts are written in:

* Python 🐍
* Bash
* Occasionally PowerShell

Example usage:

```bash
python3 script.py <target>
```

Each tool contains its own usage instructions at the top of the file.

## 🎯 Purpose of This Repo

This repository serves as:

* A personal toolkit
* A learning archive
* A reference for future machines
* A reusable automation library

---

## 🧠 Why Custom Tools?

Many Hack The Box machines require:

* Writing custom exploits
* Automating manual steps
* Bypassing filters
* Handling unique edge cases

Instead of rewriting scripts every time, this repository centralizes them.
